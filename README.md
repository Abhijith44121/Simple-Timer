# React Timer App

A simple React timer app built with hooks.

## Features

- Start and pause timer
- Reset timer to zero
- Shows elapsed time in seconds

## How to Run Locally

1. Clone the repo  
2. Navigate into the project folder  
3. Install dependencies  
4. Start the app  

## Built With

- React
- Tailwind CSS (optional styling)
